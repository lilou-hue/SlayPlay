# Flappy Bird

## Overview
A browser-based Flappy Bird clone built with vanilla HTML, CSS, and JavaScript using the Canvas API. No build system or backend required.

## Project Architecture
- `index.html` - Main HTML page with game canvas and HUD
- `script.js` - Game logic (physics, rendering, collision detection, scoring)
- `style.css` - Styling and responsive layout
- Static site served via `serve` on port 5000

## How to Run
The project runs as a static site using `npx serve -s . -l 5000`.
